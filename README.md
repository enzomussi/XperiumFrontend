# Xperium – Frontend

Frontend de un e-commerce desarrollado con React.

## Tecnologías
- React
- JavaScript
- HTML
- CSS

## Funcionalidades
- Listado de productos
- Filtros
- Carrito de compras
- Navegación con React Router
- Integración con API REST

## Proyecto académico
Proyecto realizado en equipo como parte de la Tecnicatura en Desarrollo de Software.
